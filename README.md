# F6d            
## Front-end toolkit

```javascript
npm install f6d -g
```

# Command
## init 
### Initializing project from template.

```javascript
// example
f6d init npmpkg pkgname  // Create folder('pkgname') and initializing project from template(npmpkg);
```

## init 
### Initializing project from template.

```javascript
// f6d init [repository] [project]

// example

```
